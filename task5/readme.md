# Laundry Service Hero Section

This project is a simple responsive landing page for a Laundry Service built using only HTML and CSS.

## Project Structure

```
Laundry-Hero-Section/
│── index.html
│── style.css
└── README.md
```

## Features

- Navigation bar
- Logo on the left
- Navigation links in the center
- Username section on the right
- Hero section with heading, description, button, and image
- Uses viewport (`vw` and `vh`) units
- Responsive layout using `display: inline-block`
- No Flexbox
- External CSS file

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Download or clone the project.
2. Keep `index.html`, `style.css`, and `README.md` in the same folder.
3. Open `index.html` in any modern web browser.
