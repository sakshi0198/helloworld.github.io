# Responsive Laundry Mart Landing Page

## Description

This project is a responsive landing page for Laundry Mart built using HTML5 and CSS3.

## Features

- Responsive Navigation Bar
- Flexbox Layout
- Desktop, Tablet & Mobile Views
- Hero Section
- Responsive Image
- Responsive Typography
- CTA Button
- Google Fonts (Poppins)

## Files

```
Laundry-Mart/
│── index.html
│── style.css
└── README.md
```

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries

## Responsive Breakpoints

- Desktop (>992px)
- Tablet (≤992px)
- Mobile (≤768px)

## Assignment Requirements Covered

- ✅ Flexbox used
- ✅ Responsive using Media Queries
- ✅ Navbar shrinks on tablet
- ✅ Navigation links hidden on mobile
- ✅ Hero section changes to column layout on mobile
- ✅ Responsive typography
- ✅ Responsive image
- ✅ No Bootstrap
