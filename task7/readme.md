# Responsive Laundry Website

## Description

This project is a responsive landing page for a laundry service.

I first created the desktop version using Flexbox.
After that I used media queries to make the layout work
on tablets and mobile phones.

## Learning Process

While doing this assignment I learned:

- How Flexbox arranges items in a row.
- How `justify-content` and `align-items` work.
- Why `flex-direction: column` is useful on smaller screens.
- How media queries apply different CSS based on screen width.
- How responsive images resize using percentages.

## Design Decisions

Desktop
- Navigation links are visible.
- Hero section is displayed in two columns.

Tablet
- Reduced text size.
- Reduced image size.
- Navigation text is smaller.

Mobile
- Navigation links are hidden.
- Only the logo and username are shown.
- Hero section changes from row to column.
- Button becomes full width.

## How to Run

1. Download the project folder.
2. Keep all three files in the same folder.
3. Open `index.html` in any web browser.
4. Resize the browser or use Developer Tools to test the desktop, tablet, and mobile layouts.
