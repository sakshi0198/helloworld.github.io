# Laundry Mart Responsive Website

## Description

This project is a responsive landing page for Laundry Mart.

The desktop version displays the navigation links normally.

For the mobile view, the navigation links are hidden and replaced with a hamburger menu.

The hamburger menu is created using only HTML and CSS without JavaScript.

## Features

- Responsive navigation
- Flexbox layout
- Hamburger menu
- CSS :focus-within selector
- Responsive hero section

## Files

```
index.html
style.css
README.md
```

## How to Run

1. Keep all files in one folder.
2. Open `index.html` in your browser.
3. Resize the browser window or use Developer Tools.
4. On mobile width, click the ☰ icon to display the navigation menu.

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
- CSS :focus-within
