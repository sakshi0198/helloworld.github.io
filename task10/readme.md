# Laundry Mart - Image Animation

## About the Project

This project is a simple Laundry Mart webpage created using HTML and CSS. I added an animation to the hero image to make the page look more attractive and interactive. The animation is created only with CSS, without using JavaScript.

## Project Files

```
Project Folder
│── index.html
│── style.css
└── README.md
```

## Technologies Used

- HTML
- CSS

## Features

- Responsive hero section
- Animated hero image
- Smooth floating movement
- Stretch and squeeze effect
- CSS-only animation

## How to View the Project

1. Download or clone the project.
2. Open the project folder.
3. Double-click on `index.html` or open it in any web browser.

## CSS Properties Used

- `@keyframes`
- `animation`
- `transform`
- `translate()`
- `scale()`
- `rotate()`

## Learning Outcome

While building this project, I learned:

- How to create animations using CSS.
- How to move elements with `translate()`.
- How to combine `rotate()` and `scale()` in one animation.
- How to make a webpage look more interactive without JavaScript.
