# CSS Assignment

This project demonstrates the use of HTML and CSS selectors.

## Files

```
Project Folder/
│── index.html
│── style.css
└── README.md
```

## Features

- External CSS linked to HTML
- Background color using CSS
- Heading styling using class selector
- Background styling using ID selector
- Text styling using span selector
- Paragraph styling using element selector
- Customized paragraph using ID selector
- Styled hyperlinks without using inline CSS
- Styled ordered list items using CSS selectors

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Download the project folder.
2. Open `index.html` in any web browser.

## Output

The webpage displays:
- Three headings
- Three paragraphs
- Two hyperlinks
- One unordered list
- One ordered list
- Styling entirely using an external CSS file.
