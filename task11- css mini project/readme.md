# Thala 7 Sneaker Store

## About the Project

This is a front-end sneaker store website created using **HTML** and **CSS**. The project is designed as a mini e-commerce website where users can browse shoes, view product details, check their shopping cart, and proceed to the payment page.

The entire project is built using **Flexbox** for layout and **CSS3** for styling. No frameworks like Bootstrap or JavaScript have been used.

---

## Pages Included

### Landing Page
- Navigation bar with logo, menu, search, cart, and profile icon
- Hero section with shoe image and promotional content
- "Shop Now" button
- Service section showing reviews, customers, quality, and ratings

### Product Display Page
- Large product image
- Product name and description
- Price and discount details
- Quantity selector
- Add to Cart button
- Scrollable image gallery

### Shopping Cart Page
- Shopping cart table
- Product quantity and price
- Remove product option
- Coupon code section
- Order summary
- Pay Now button
- Continue Shopping link

### Payment Page
- Contact details form
- Payment method selection
- Card payment form
- Order summary
- Coupon code option
- Pay Now button
