# CSS Task 4 - Units, Box Model & Fonts

This project demonstrates the use of CSS viewport units, box model, and button styling.

## Project Structure

```
Task4/
│── index.html
│── style.css
└── README.md
```

## Features

- External CSS file
- Responsive layout using viewport units (`vw` and `vh`)
- Image displayed inside a centered container
- Gradient button
- Proper use of `box-sizing`
- No Flexbox
- No absolute/relative positioning
- No `overflow: hidden`

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Download the project folder.
2. Open `index.html` in any modern web browser.

## Expected Output

- Image centered on the page.
- Equal spacing of 25% from left and right.
- 10% spacing from top and bottom.
- Gradient "Click Me" button below the image.
