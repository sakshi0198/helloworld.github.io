# Laundry Mart Login Page

This project is a responsive Laundry Mart Login Page built using HTML5 and CSS3 with Flexbox.

## Project Structure

```
Laundry-Mart-Login/
│── index.html
│── index.css
└── README.md
```

## Features

- Split-screen login page using Flexbox
- Laundry illustration on the left
- Login form on the right
- Email and password fields
- Login button
- Forgot Password link
- Register link
- Modern and clean UI
- Responsive layout

## Technologies Used

- HTML5
- CSS3 (Flexbox)

## How to Run

1. Save the files in the same folder:
   - `index.html`
   - `index.css`
   - `README.md`
2. Open `index.html` in any modern web browser.
