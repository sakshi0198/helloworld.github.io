# Laundry Wallah / Laundry Mart

This project is a simple HTML webpage created as part of the Laundry Wallah / Laundry Mart assignment.

## Project Structure

```
LaundryWallah_Task/
│── index.html
└── README.md
```

## Features

- Welcome heading
- About section
- Laundry image
- List of services
- Price table
- Booking form with required fields
- Footer with contact information

## Technologies Used

- HTML5

## How to Run

1. Download or clone the project.
2. Open the `index.html` file in any web browser.

## Expected Output

The webpage displays:
- Welcome heading
- About Laundry Wallah / Laundry Mart
- Laundry image
- Services list
- Price table
- Booking form
- Contact information in the footer
